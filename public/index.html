Social Media App

This project is a fully functional social media web application developed using the Django framework. It allows users to register, log in, create and manage posts, like, comment, and interact with other users through a user-friendly interface. The app features user authentication, dynamic newsfeeds, follow/unfollow systems, and media upload support. A responsive UI was built using HTML, CSS, and Bootstrap, ensuring cross-device compatibility. Django’s admin panel was utilized for backend management of users, posts, and comments. This project enhanced my skills in web development, Django ORM, relational database design, and user interaction logic.

Significant amounts of edible food are wasted by households, restaurants, caterers, and events, while many people go hungry. Simultaneously, organic waste (including unavoidable food waste) is often not managed efficiently, leading to environmental problems.Develop a web-based platform using ASP.NET Web Forms that facilitates the reduction, redistribution, and efficient management of food waste across different stakeholders in Ahmedabad.
